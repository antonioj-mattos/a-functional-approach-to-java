# Chapter 14: (Functional) Design Patterns

The examples of this chapter are designed as a Gradle project.

## Factory Pattern

`./gradlew factoryOO`  
`./gradlew factoryFP`

## Decorator Pattern

`./gradlew decoratorOO`  
`./gradlew decoratorFP`

## Strategy Pattern

`./gradlew strategyOO`  
`./gradlew strategyFP`

## Builder Pattern

`./gradlew builderOO`  
`./gradlew builderFP`
